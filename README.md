# Snake
Классическая игра «Змейка» (Snake) на 7 языках программирования. Каждая версия поддерживает:

- Управление стрелками или WASD
- Змейка растёт при поедании еды
- Счёт и рекорд (хранится локально)
- Ускорение с ростом длины
- Столкновение с собой или стенами (с выбором режима)
- Пауза (пробел или P)
- Звуковые эффекты (опционально)

## Реализации

| Язык | Технология | Особенности |
|------|------------|-------------|
| Python | Pygame | Графика, звук, рекорды |
| Go | Ebitengine | Лёгкий 2D-движок |
| Rust | ggez | Высокая производительность |
| C# | Windows Forms | Классический Windows |
| JavaScript | HTML5 Canvas | Браузерная версия |
| TypeScript | Canvas + классы | Типизированная версия |
| Java | Swing | Кроссплатформенный GUI |
